Codex皮肤主题管理器 Windows x64 版

系统要求：
1. Windows 10 或 Windows 11（x64）。
2. 已从 Microsoft Store 安装并至少启动过一次 OpenAI Codex。
3. 无需另装 Node.js，压缩包已包含经过官方校验的 Node.js 运行时。

首次使用：
1. 完整解压 ZIP，不能直接在压缩包内运行。
2. 双击“安装并启动皮肤.cmd”。如果 Codex 已打开，会先询问是否重启。
3. 安装完成后，运行“Codex皮肤主题管理器.exe”即可管理皮肤。

皮肤管理：
- 点击“导入皮肤”或把 .codexskin 文件拖入管理器。
- 选择皮肤后点击“应用并启动”，Codex 会切换到该皮肤。
- 选择任意皮肤后点击“重命名”，或按 F2 修改显示名称；皮肤 ID 和当前启用状态不会改变。
- “导出皮肤包”可生成模板，文件规范见“皮肤包格式.txt”。
- 导入的皮肤保存在 %LOCALAPPDATA%\CodexDreamSkin\skins。

日常使用：
- 请通过“Codex Dream Skin”快捷方式或本目录的“启动皮肤.cmd”启动。
- 直接点击官方 Codex 图标不会加载皮肤。
- Codex 自动更新后，再运行一次“安装并启动皮肤.cmd”。

验证与恢复：
- “验证皮肤.cmd”会在桌面生成验证截图。
- “恢复官方外观.cmd”会关闭皮肤调试会话、恢复安装前的外观设置并删除皮肤快捷方式。

注意：
- 请保留整个解压目录，桌面快捷方式会引用其中的运行引擎。
- 皮肤仅监听本机 127.0.0.1 调试端口，不修改 WindowsApps 或 app.asar。
