# Codex Skin Manager

Codex Skin Manager 是一个面向 Windows 的 Codex 皮肤管理器，用于导入、预览、切换和恢复 Codex 桌面端外观。

## 功能

- 导入 `.codexskin` 或 `.zip` 皮肤包。
- 预览皮肤背景、人像图和主题信息。
- 一键应用当前皮肤并启动或热更新 Codex。
- 恢复 Codex 官方外观。
- 重命名、导出、删除本地皮肤。
- 内置多套人像主题皮肤。
- 提供“星光动态”开关，可开启或关闭会话背景动态星光特效。
- 提供 Windows 安装包，默认安装到 `E:\codex-skin-manager`。

## 二次引用说明

本项目基于 [Codex Dream Skin](https://github.com/Fei-Away/Codex-Dream-Skin) 进行二次开发和封装，保留其 Codex 皮肤注入、主题运行时与部分皮肤结构设计，并在此基础上增加 Windows 皮肤管理器、安装器、内置皮肤、动态特效开关和更适合分发的本地安装体验。
